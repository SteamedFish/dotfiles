## English

Personal dotfiles managed with [yadm](https://yadm.io/).

### What's Inside
- Shell configs (bash/zsh) - `.bashrc.d/`
- Emacs config - `.config/emacs/`
- Neovim (LazyVim) - `.config/nvim/`
- 40+ app configs - `.config/`

### Quick Start
```bash
# Clone with yadm
yadm clone <your-repo-url>

# Reload shell
source ~/.bashrc  # or ~/.zshrc
```

### Note
This is a **highly personal** configuration. Use at your own risk.

For detailed documentation, see [AGENTS.md](./AGENTS.md).
