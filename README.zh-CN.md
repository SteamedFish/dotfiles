# 个人 Dotfiles

个人 dotfiles，使用 [yadm](https://yadm.io/) 管理。

## 包含内容

- Shell 配置（bash/zsh）: `.bashrc.d/`
- Emacs 配置: `.config/emacs/`
- Neovim（LazyVim）: `.config/nvim/`
- 40+ 应用配置: `.config/`
- Agent 技能库: `.config/agents/skills/`

## 快速开始

```bash
# 使用 yadm 克隆
yadm clone <你的仓库地址>

# 重载 shell
source ~/.bashrc  # 或 source ~/.zshrc
```

## 说明

这是一个高度个人化的配置仓库，请按需选择性复用。

更多细节见 [AGENTS.md](./AGENTS.md)。
